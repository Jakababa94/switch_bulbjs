# Bulb Light JavaScript

A simple interactive web application that simulates turning a light bulb on and off using vanilla JavaScript.

## Features

- **Interactive UI**: Click buttons to toggle between ON and OFF states
- **Visual Feedback**: Displays different bulb images based on the current state
- **Responsive Design**: Clean, centered layout with a modern card-style interface
- **Vanilla JavaScript**: No dependencies required

## Project Structure

```
├── index.html      # Main HTML file with structure
├── script.js       # JavaScript functionality for button interactions
├── style.css       # Styling and layout
├── bulb-on.jpg     # Image of lit bulb (required)
├── bulb-off.jpg    # Image of unlit bulb (required)
└── README.md       # This file
```

## How It Works

1. The application displays an image of a light bulb
2. Two buttons (`ON` and `OFF`) control the bulb state
3. Clicking `ON` changes the image to `bulb-on.jpg`
4. Clicking `OFF` changes the image to `bulb-off.jpg`

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge, etc.)

### Installation

1. Clone or download the project files
2. Ensure `bulb-on.jpg` and `bulb-off.jpg` images are in the project directory
3. Open `index.html` in your web browser

## Usage

Simply open the `index.html` file in a web browser and click the `ON` and `OFF` buttons to toggle the bulb state.

## Technologies Used

- **HTML5**: Page structure
- **CSS3**: Styling and layout
- **JavaScript (ES6)**: Interactive functionality

## Styling Details

- Background color: Light beige (#f8f4ec)
- Card background: White with subtle shadow
- ON button: Green
- OFF button: Red
- Hover effect: Slight opacity change and scale transform

## Notes

- The project requires two image files: `bulb-on.jpg` and `bulb-off.jpg`
- The application starts with the bulb image set to `bulb-off.jpg` by default
- No external libraries or frameworks are used
